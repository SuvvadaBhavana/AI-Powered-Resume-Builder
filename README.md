# 🤖 AI Powered Resume Builder

A professional AI-powered resume builder that helps you create stunning resumes with multiple templates, real-time preview, and PDF export.

## 🌐 Live Demo
👉 https://ai-powered-resume-builder-sg2l.onrender.com

## 📸 Screenshots
![App Screenshot](Screenshot%20.png)

## ✨ Features
- 📝 Fill in your details with a clean, easy-to-use form
- 🎨 3 professional templates (Classic, Modern, Minimal)
- 🤖 AI-powered content suggestions
- 📄 Download resume as PDF
- 💾 Auto-save to localStorage
- ✏️ Inline editing on generated resume
- 📱 Responsive design
- 🎯 ATS (Applicant Tracking System) score analyzer for your resume

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **Package Manager:** pnpm workspaces
- **Deployment:** Render

## 🚀 Run Locally
```bash
git clone https://github.com/SuvvadaBhavana/AI-Powered-Resume-Builder.git
cd AI-Powered-Resume-Builder
pnpm install
pnpm --filter @workspace/resume-builder run dev
```
